# RW Research Data 来源证据

## 公开来源用途

- `fair`：FAIR Principles。检查研究材料是否可发现、可取得、可互操作和可复用；FAIR 不等于公开，也不替代伦理和访问控制。 官方页面：https://www.go-fair.org/fair-principles/
- `datacite46`：DataCite Metadata Schema 4.6。检查持久标识符、版本、权利、描述和相关资源等研究对象元数据。 官方页面：https://datacite-metadata-schema.readthedocs.io/en/4.6/properties/overview/
- `icmje`：ICMJE Recommendations。作者责任、披露、投稿、重复发表和 AI 使用。 官方页面：https://www.icmje.org/recommendations/
- `cope`：COPE Core Practices。处理作者、利益冲突、数据、同行评审和发表伦理。 官方页面：https://publicationethics.org/core-practices
