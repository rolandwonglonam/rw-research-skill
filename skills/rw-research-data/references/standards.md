# RW Research Data 公开标准

这些条目用于选择方法和核验当前版本，不替代原始标准。

## FAIR Principles

- 官方页面：https://www.go-fair.org/fair-principles/
- 本 Skill 用途：检查研究材料是否可发现、可取得、可互操作和可复用；FAIR 不等于公开，也不替代伦理和访问控制。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## DataCite Metadata Schema 4.6

- 官方页面：https://datacite-metadata-schema.readthedocs.io/en/4.6/properties/overview/
- 本 Skill 用途：检查持久标识符、版本、权利、描述和相关资源等研究对象元数据。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## ICMJE Recommendations

- 官方页面：https://www.icmje.org/recommendations/
- 本 Skill 用途：作者责任、披露、投稿、重复发表和 AI 使用。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## COPE Core Practices

- 官方页面：https://publicationethics.org/core-practices
- 本 Skill 用途：处理作者、利益冲突、数据、同行评审和发表伦理。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。
