# RW Citation Audit 公开标准

这些条目用于选择方法和核验当前版本，不替代原始标准。

## Crossref REST API

- 官方页面：https://www.crossref.org/documentation/retrieve-metadata/rest-api/
- 本 Skill 用途：核验 DOI 和出版元数据；成员提交的元数据可能不完整。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## ORCID Public API

- 官方页面：https://info.orcid.org/what-is-orcid/services/public-api/
- 本 Skill 用途：核验公开作者标识和公开记录。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## ICMJE Recommendations

- 官方页面：https://www.icmje.org/recommendations/
- 本 Skill 用途：作者责任、披露、投稿、重复发表和 AI 使用。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。
