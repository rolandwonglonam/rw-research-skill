# RW Citation Audit 来源证据

## 公开来源用途

- `crossref`：Crossref REST API。核验 DOI 和出版元数据；成员提交的元数据可能不完整。 官方页面：https://www.crossref.org/documentation/retrieve-metadata/rest-api/
- `orcid`：ORCID Public API。核验公开作者标识和公开记录。 官方页面：https://info.orcid.org/what-is-orcid/services/public-api/
- `icmje`：ICMJE Recommendations。作者责任、披露、投稿、重复发表和 AI 使用。 官方页面：https://www.icmje.org/recommendations/

## 边界

- 这里保存的是来源定位和短用途说明，不是完整标准副本。
- 报告规范不等于设计质量评价；偏倚工具不等于写作模板。
- 版本、政策、费用、API 和期刊要求在任务执行时核验。
