# RW PhD Tone 选择表

| 情形 | 执行动作 | 不要做 |
|---|---|---|
| 有多段本人同类章节 | 提取重复模式并标置信心 | 逐句照抄 |
| 只有当前草稿 | 建立临时基线并说明限制 | 称为稳定指纹 |
| 只有外部范文 | 只提取体裁移动 | 当成用户声音 |
| 语气和准确性冲突 | 先保留准确性 | 为了像作者而保留错误 |
| 修改范围较大 | 先给偏移说明和局部示例 | 静默重写整章 |

## 核心规则

- 作者指纹必须来自用户明确提供或指定的作者语料。
- 用户指定的持久 tone profile 优先于本轮临时推断；profile 不可用时说明并回到当前样本。
- 外部优秀论文只能提供体裁和章节移动参考。
- 没有作者样本时，输出中性学术注册，不能声称复现个人声音。
- 语气规则不能覆盖事实、数字、引用和方法边界。
- 单个句子不能形成稳定的作者规则。
- 规则要说明适用章节，Methods 和 Discussion 不能共用全部规则。
- 词频、句长和句法指标只提供线索，不直接等于声音。
- 保留作者常用的限定、犹豫和让步，不把它们全部删掉。
- 不把所有句子改成同一长度和节奏。
- 修改优先处理偏移最大的句子，不整篇重写。
- 语法错误修复和作者声音修改要分开记录。
- 每条语气判断都要能指向一个或多个样本位置。
