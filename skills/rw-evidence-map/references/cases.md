# RW Evidence Map 案例和反例

> 公开案例使用合成或占位输入，不来自任何个人研究项目。

## 案例 1

- 输入：比较 3 种测量方法的证据。
- 必须做：定义证据图服务的研究问题、决策和时间范围。；输出证据簇、冲突原因、空白和下一步取证。；证据矩阵或关系图。
- 不能做：依赖未提供的本地工作区内容；生成不存在的论文、数据、DOI、工具结果或期刊要求；越过停止条件：摘要不足以判断方法或结果时，标记待全文核对。
- 下一步：rw-research-design。
