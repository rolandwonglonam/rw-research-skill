# 方法来源

## 包内规则

- 2026-07-13：建立独立的 `RW Research Passport`。
- 一个项目只保存自己的材料、判断、未知项和交接状态。
- 原始材料留在原位，状态文件只保存指针和核验状态。

## RW 系统关系

- `rw-save`：保存阶段状态和下一步。
- `rw-paper-extractor`：提供材料 ID、结构化字段和原文位置。
- `rw-evidence-map`：连接研究、结果、冲突和证据关系。
- `rw-phd-write`：接收已确认材料进入写作。

## 实现

- JSON 结构和 Python 脚本随 Skill 提供。
- 不读取跨项目个人记忆，不复制论文全文，不上传研究材料。
