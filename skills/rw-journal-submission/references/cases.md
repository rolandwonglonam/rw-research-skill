# RW Journal Submission 案例和反例

> 公开案例使用合成或占位输入，不来自任何个人研究项目。

## 案例 1

- 输入：为一篇 scoping review 比较 3 本期刊。
- 必须做：确认论文类型、研究范围、方法、目标读者、作者和投稿限制。；逐条记录处理动作、修改位置、证据、不同意理由和待作者确认项，再核对回复信与修改稿。；带官方来源和日期的期刊比较。
- 不能做：依赖未提供的本地工作区内容；生成不存在的论文、数据、DOI、工具结果或期刊要求；越过停止条件：无法访问期刊当前官方要求时，不标记投稿准备完成。
- 下一步：rw-phd-write。
