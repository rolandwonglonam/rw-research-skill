# RW PhD Write 公理

## AXIOM-01：写作从章节功能和主张开始

- 规则：写作从章节功能和主张开始，不从漂亮句子开始。
- 来源原子：`RWS-PHD-WRITE-008`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有来源时，不生成事实性引用或伪造 DOI。

## AXIOM-02：每个事实性主张都要有用户材料或可核验来源

- 规则：每个事实性主张都要有用户材料或可核验来源。
- 来源原子：`RWS-PHD-WRITE-009`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有研究结果时，不写成研究已完成。

## AXIOM-03：来源支持范围小于句子范围时

- 规则：来源支持范围小于句子范围时，收窄句子。
- 来源原子：`RWS-PHD-WRITE-010`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不为流畅删除限制、反例和不确定性。

## AXIOM-04：结果、解释、推断和建议要使用不同的证据强度

- 规则：结果、解释、推断和建议要使用不同的证据强度。
- 来源原子：`RWS-PHD-WRITE-011`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不把 Skill 输出当成作者最终责任或期刊合规确认。

## AXIOM-05：Introduction 建立问题和缺口

- 规则：Introduction 建立问题和缺口，不能提前假定研究结论。
- 来源原子：`RWS-PHD-WRITE-012`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有来源时，不生成事实性引用或伪造 DOI。

## AXIOM-06：Methods 说明做了什么和为何这样做

- 规则：Methods 说明做了什么和为何这样做，不能用结果合理化方法。
- 来源原子：`RWS-PHD-WRITE-013`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有研究结果时，不写成研究已完成。

## AXIOM-07：Results 报告预定和实际分析

- 规则：Results 报告预定和实际分析，不用 Discussion 语言解释。
- 来源原子：`RWS-PHD-WRITE-014`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不为流畅删除限制、反例和不确定性。

## AXIOM-08：Discussion 先回答研究问题

- 规则：Discussion 先回答研究问题，再比较证据、解释限制和外推边界。
- 来源原子：`RWS-PHD-WRITE-015`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不把 Skill 输出当成作者最终责任或期刊合规确认。

## AXIOM-09：章节编号不是写作功能

- 规则：根据内容、研究问题和上下文判断功能，不按 Chapter 编号套模式。
- 来源原子：`RWS-PHD-WRITE-029`、`RWS-PHD-WRITE-030`。
- 适用：论文、研究计划和科研报告中的任何章节、小节或段落。
- 停止：缺少正文和上下文时，只提出候选功能。

## AXIOM-10：解释义务随功能变化

- 规则：每个部分都要完成自己的功能，但不机械加入 evidence、interpretation、why it matters 和 research-question link。
- 来源原子：`RWS-PHD-WRITE-031`、`RWS-PHD-WRITE-038`。
- 适用：判断段落是否需要补充解释、重要性或研究问题连接。
- 停止：Results 不被强行写成 Discussion。

## AXIOM-11：关键推理不能交给读者补全

- 规则：补写有证据支持的关键前提、解释和后果；没有材料时标记缺口。
- 来源原子：`RWS-PHD-WRITE-032`、`RWS-PHD-WRITE-033`、`RWS-PHD-WRITE-039`。
- 适用：连续引用、研究缺口、研究意义和导师表示不理解的段落。
- 停止：不生成通用意义句或未被材料支持的解释。

## AXIOM-12：具体性来自研究材料

- 规则：保留样本、测量、比较、数字、效应、不确定性和引用位置。
- 来源原子：`RWS-PHD-WRITE-034`。
- 适用：扩写、增加细节、提高严谨度或按正向样本修订。
- 停止：材料没有提供时，不补造研究细节。

## AXIOM-13：正向样本校准论证

- 规则：从被肯定的段落提取写作功能、证据密度、解释深度和句间连接，并保持已经成立的部分。
- 来源原子：`RWS-PHD-WRITE-036`、`RWS-PHD-WRITE-037`。
- 适用：用户提供导师或审稿人的正向评价和修改批注。
- 停止：样本不足时，只报告可观察特征。
