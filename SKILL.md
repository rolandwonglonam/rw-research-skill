# RW Research Skill

这个包包含 20 个科研 Skill。它处理本地研究积累扫描、学习起点、研究问题、文献发现、论文提取、证据关系、研究设计、研究材料、统计报告、引用核验、主张核验、写作、局部修订、投稿和项目状态。

## 使用方式

第一次使用时，先调用 `rw-research-router`。用户不知道从哪里开始时，它会进入 `rw-research-learning`，扫描指定文件夹或当前可读项目，判断可见研究基础并选择第一步。

## 运行边界

- 用户提供的材料是当前任务输入。
- 需要当前文献、期刊政策、工具状态或 API 信息时，回到官方来源核验。
- 不生成不存在的论文、数据、DOI、期刊要求或工具结果。
- 涉及临床、伦理或统计决定时，不把本包作为专业审批。

完整的 20 个 Skill、安装方式和验证记录见 `README.md`。
