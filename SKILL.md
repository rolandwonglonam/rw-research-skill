# RW Research Skill

这个包包含 19 个科研 Skill。它处理研究问题、文献发现、论文提取、证据关系、研究设计、研究材料、统计报告、引用核验、主张核验、写作、局部修订、投稿和项目状态。

## 使用方式

第一次使用时，先调用 `rw-research-router`。提交你手头的研究想法、文献、数据、草稿、审稿意见，或直接说明卡在哪里。它会选择当前需要的一个 Skill，不会一次启动全部流程。

## 运行边界

- 用户提供的材料是当前任务输入。
- 需要当前文献、期刊政策、工具状态或 API 信息时，回到官方来源核验。
- 不生成不存在的论文、数据、DOI、期刊要求或工具结果。
- 涉及临床、伦理或统计决定时，不把本包作为专业审批。

完整的 19 个 Skill、安装方式和验证记录见 `README.md`。
