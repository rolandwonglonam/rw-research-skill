# 公共内容边界

公开发行包只保存通用研究方法、公开来源、包内规则和合成测试输入。

## 不进入公开包

- 姓名、邮箱、账号、设备路径和组织内路径。
- 个人论文、数据、评审记录、导师反馈和研究项目状态。
- 能连接到具体研究主题、样本、人群、机构、项目编号或伦理编号的组合线索。
- 能指向个人选择、个人反馈或本地设计记录的来源标签。
- 知识原子中的原始材料摘录。

## 公开内容规则

- 知识原子只保存当前规则和公开来源标识。
- 包内方法使用 `packaged_*` 来源标识。
- 跨 Skill 规则使用 `rw-*` 来源标识。
- 案例和行为合同使用合成或占位输入，不取自个人研究项目。
- 公开来源保留名称和链接，但不加入个人使用记录。

## 构建门

`scripts/check_public_privacy.py` 检查公开文件、469 条知识原子、115 个案例和 116 条行为合同。检查失败时不构建发行包。
